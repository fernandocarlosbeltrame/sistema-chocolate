# 🍫 Sistema de Vendas de Chocolates

Sistema completo de vendas desenvolvido em Python com Streamlit.

## 🚀 Funcionalidades

- Cadastro de vendas
- Controle de clientes
- Controle de fiado (pendente/pago)
- Cadastro de produtos
- Edição e exclusão de vendas
- Dashboard com gráficos
- Controle de estoque automático

## 💻 Tecnologias

- Python
- Streamlit
- Pandas

## 📊 Objetivo

Sistema criado para controle de vendas de pequenos negócios.
